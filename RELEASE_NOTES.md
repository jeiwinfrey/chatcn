# First Public Release

## chatcn 0.1.0

chatcn is a CLI that scaffolds production-ready AI chatbot templates into existing shadcn projects.

### Highlights

- Five chatbot templates, including a custom starter
- Interactive provider and model selection
- Beginner-friendly setup flow
- Markdown rendering and loading states in the generated UIs
- Framework-aware API route generation for supported stacks
- Public npm and GitHub release metadata

### Notes

- The CLI requires an existing shadcn project
- Supported frameworks include Next.js, Remix, Astro, TanStack Start, React Router v7, Laravel, and Vite
- Vite projects require a manually wired backend route
